1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Use constants (const) when the value should never change during the program execution (e.g., const pi = 3.14). Use variables (let) for values that might change (e.g., user input, counters). Not all values can be constants because some need to change depending on program logic.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses change the precedence of operations in an expression. Without parentheses, operations follow the default precedence (multiplication/division > addition/subtraction). Adding parentheses lets you control the order in which parts of the expression are evaluated.
    Example: (2 + 3) * 4 = 20 vs. 2 + 3 * 4 = 14.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. Use the .toFixed() method. For example:
    let value = 3.14159; value.toFixed(2) limits the value to 3.14. This is useful for formatting outputs.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. Compare them using a small tolerance value to account for floating-point precision issues:
    Math.abs(12.199999 - 12.2) < 0.00001 (returns true).

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return a Boolean value (true or false). You can check their result directly in conditions or log them to the console.
    Example: console.log(5 > 3); // true, console.log(2 === 3); // false.

  6. How does the Math.random() function work, and what can it be used for?

    7. The Math.random() function generates a random decimal number between 0 (inclusive) and 1 (exclusive). You can scale it to generate numbers in a range.
    Example: Math.floor(Math.random() * 10) + 1 generates a random number between 1 and 10.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus operator (%) returns the remainder of a division.
    Example: 10 % 3 = 1 (10 divided by 3 leaves a remainder of 1).
    It's commonly used for tasks like determining even/odd numbers : if (x % 2 === 0), cycling through ranges, and finding remainders.
