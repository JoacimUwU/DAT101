1. 1.What is an array?

    2. An array is a data structure that can store multiple values under a single name. Each value can be accessed by its index (position in the array), starting from zero.

  2. 2.What does it mean to "hardcode" in programming?

    3. Hardcoding refers to directly embedding data into the source code instead of dynamically generating or retrieving it during runtime. This makes the data less flexible and harder to update.
