1. What is an array?

    2. An array is a data structure that can store multiple values under a single name. Each value can be accessed by its index (position in the array), starting from zero.
    Example: let numbers = [1, 2, 3];

  2. What does it mean to "hardcode" when we talk about programming?

    3. Hardcoding refers to embedding fixed values directly into the source code instead of dynamically generating or retrieving them during runtime.
    Example: let colors = ["red", "green", "blue"];

  6. In Task 8, we will use a "callback" routine when sorting. Explain how such a "callback" routine works.

    7. A callback function in sorting determines the order of elements. The function compares two elements and returns:

    • A negative value if the first element comes before the second.
    • A positive value if the first element comes after the second.
    • Zero if the elements are equal.
    Example: array.sort((a, b) => a - b); // Sorts numbers in ascending order.
